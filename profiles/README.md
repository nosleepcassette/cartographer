# Profiles

Cartographer ships with two wire vocabularies:

- `default` keeps things neutral and general-purpose.
- `emotional-topology` is for atlases where the graph needs to reflect lived emotional structure, not just abstract links.

`emotional-topology` was built for a brain with ADHD, autism, and real avoidance patterns. It is not a metaphor. Use it if that resonates.

Profiles only apply when you initialize a new atlas or when you explicitly run `cart profile apply <name>`. Existing atlases keep their current behavior unless you switch them on purpose.
